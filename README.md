## Overview

LemurSystem is a comprehensive, web-based ERP (Enterprise Resource Planning) solution designed specifically for small to medium enterprises. It provides integrated modules for managing all aspects of your business operations.

## Vision

To simplify business operations by providing a unified platform that reduces communication overhead and streamlines workflow processes.

---

## Features

### Core Modules

1. **Human Resources**
   - Employee management and directory
   - Leave tracking and approval workflows
   - Resume upload and storage
   - Employee ID card generation
   - Department management

2. **Payroll Management**
   - Automated salary processing
   - Tax calculations and deductions
   - Bonus and benefits management
   - Payment history tracking

3. **Financial Services**
   - Professional invoice creation
   - PDF invoice generation
   - Accounts payable and receivable
   - Financial reports
   - CSV export

4. **Supply Chain**
   - Inventory management
   - Low stock alerts
   - Vendor management
   - Stock valuation

5. **CRM (Customer Relationship Management)**
   - Customer database
   - Lead tracking and pipeline
   - Sales analytics
   - Customer ID cards

### Key Features

- **Cloud-based**: Access from anywhere
- **Responsive**: Works on desktop, tablet, and mobile
- **Secure**: Enterprise-grade security
- **Fast**: Optimized for performance
- **Scalable**: Grows with your business

## Quick Start

### Prerequisites

- Node.js 18+ (for local development)

### Installation

```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Access

- **Web Application**: http://localhost:3000
- **Documentation**: http://localhost:3000/docs

### Getting Started

1. Open the application
2. Click "Start Free Trial" or go to /login
3. Register with your organization details
4. Select your industry
5. Choose your pricing plan
6. Access the dashboard
